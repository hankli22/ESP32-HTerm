# ESP32-HTerm

![GitHub release](https://img.shields.io/github/v/release/hankli22/ESP32-HTerm?color=blue&logo=github)
![License](https://img.shields.io/github/license/hankli22/ESP32-HTerm?color=green)

An advanced 66FPS Handheld Terminal for LR Sensors and Hypixel API tracking.

### **v1.0.0 Changelog**
- Added Hypixel API support with real-time tracking.
- Implemented UI Culling logic for 66FPS performance.
- Fixed WiFi/ESP-NOW hardware conflicts.

### **Usage**
Replace placeholders in `secret.cpp` with your actual credentials before flashing.
